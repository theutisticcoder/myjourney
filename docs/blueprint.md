# **App Name**: MyJourneyAI

## Core Features:

- Speed Tracking and Simulation: Track the user's running/walking speed using location data and a manual MPH simulator slider.
- AI-Generated Story Adaptation: Dynamically generate a story that adapts its narrative tone to the user's physical intensity, using speed to adjust the story's stylistic mandates. Includes a tool which decides whether or not to use that piece of data
- Chapter Generation and Pacing: Trigger a new chapter every ten minutes or one mile based on user preferences, adapting chapter generation timing based on selected carpool settings.
- Session State Persistence: Store the entire session state (full story context, generated text, audio Blobs) to allow users to resume their adventure exactly where they left off.
- Saved Stories Dashboard: Provide a dashboard to access and resume saved stories, maintaining full history and voice narration.
- Text-to-Speech Narration: Convert the generated text into realistic TTS audio.
- Real-time Metrics and Controls: Display real-time dashboard metrics (Speed, Distance, CO2 emissions saved) with a bottom control overlay containing a workout timer and session management tools.

## Style Guidelines:

- Primary color: Electric Blue (#7DF9FF) for a vibrant, futuristic feel.
- Background color: Dark grey (#212529) to emphasize the cyberpunk theme with a dark scheme.
- Accent color: Neon Purple (#CF6679) for interactive elements and highlights.
- Body text and headline font: 'Space Grotesk', sans-serif font, for a modern and technical look.
- Code font: 'Source Code Pro' for session management and controls display.
- Literary-style chapter feed with drop-cap typography and loading shimmer effects for an immersive reading experience.
- Use neon-outlined icons to represent genres and workout options.
